## Contexto Academico

Compilador que traduce instrucciones simplificadas escritas en formato natural a bloques visuales de Scratch representados en JSON.

## Proposito

- Facilitar la creacion de proyectos en Scratch mediante instrucciones textuales.
- Aplicar conceptos de gramatica, parsing y generacion de salida.
- Practicar Java y ANTLR.
- Conservar evidencia academica de proyecto integrador.

- `grammar/`: gramatica ANTLR.
- `src/`: codigo fuente principal y generadores.
- `salida/`: resultados generados en JSON.
